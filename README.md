<h4>How to Play</h4>
<p>
  1. Choose "New Game"<br/>
  2. Investigate your board<br/>
  3. Choose between three different flipping patterns<br/>
  4. Flip all the tiles to red<br/>
  5. Level up!<br/>
  6. See how far you can get. Can you beat all the levels?<br/>
